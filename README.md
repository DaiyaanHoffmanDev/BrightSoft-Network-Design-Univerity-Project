# BrightSoft-Network-Design-Univerity-Project
Secure 3-floor enterprise network for a software firm.  Floor 1: Legacy RIPv2 &amp; local DHCP.  Floor 2: OSPF, VLANs, &amp; Inter-VLAN routing (L3 Switch).  Floor 3: Secured Staging DMZ via Extended ACLs.  Resilience: Redundant failover links. Designed for high security and scalability.
