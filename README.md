- BrightSoft-Network-Design-Univerity-Project
Secure 3-floor enterprise network for a software firm.  Floor 1: Legacy RIPv2 &amp; local DHCP.  Floor 2: OSPF, VLANs, &amp; Inter-VLAN routing (L3 Switch).  Floor 3: Secured Staging DMZ via Extended ACLs.  Resilience: Redundant failover links. Designed for high security and scalability.
- BrightSoft Solutions Network Design

-Project Overview
Secure 3-floor enterprise network simulation built in Cisco Packet Tracer.

-Key Features
-Hybrid Routing:** RIPv2 (Floor 1) and OSPF (Floor 2).
-Segmentation:** VLANs for Test & UI teams with Inter-VLAN routing.
-Security:** Extended ACLs protecting the Floor 3 Staging DMZ.
-Resilience:** Failover backup link between BA and Shared routers.

-Verification
-DHCP successfully assigns IPs across all subnets.
-ACLs permit Developer updates but block unauthorized internal access to Staging.
